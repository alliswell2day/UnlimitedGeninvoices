# UnlimitedGeninvoices
### A geninvoice plugin for WHMCS
### WHMCS提前续费账单生成插件

## 用法
* 1.上传到/module/addons
* 2.激活插件
* 3.在主页加一个链接 /index.php?m=UnlimitedGeninvoices

## Usage
* 1.Upload to /module/addons
* 2.active the plugin
* 3.make a link to /index.php?m=UnlimitedGeninvoices
